# Ghoomo User App

Student/User app for requesting rides and tracking drivers.

## Features

- Login with IIT email (OTP)
- Request rides
- Track driver location
- View ride history
- Emergency alerts

## Getting Started

```bash
npm install
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000).

## Project Structure

```
src/
├── components/     # React components
├── pages/         # Page components
├── services/      # API services
├── hooks/         # Custom hooks
├── styles/        # CSS files
└── App.js         # Main app component
```

## Available Scripts

- `npm start` - Run development server
- `npm build` - Build for production
- `npm test` - Run tests
